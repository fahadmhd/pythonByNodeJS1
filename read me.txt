Beginners can follow the steps below:

1) make directory or folder with some name
mkdir pythonByNodeJS
2) go to it by command prompt
cd pythonByNodeJS
3) install express framework
npm init
npm i express
4) write hello world in python in file name, e.g., script1.py
 print('Hello World from python')
5) create a index.js file and copy paste the code from the attached file
        OR   download attachment 
6) run index.js by the following command
node index.js
